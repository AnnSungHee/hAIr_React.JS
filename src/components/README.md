# components
- 이 디렉토리는 웹서비스에 사용되는 컴포넌트들을 모아놓은 디렉토리입니다.

- `BotChatComponent`: 채팅페이지에서 사용되는 AI봇의 말풍선 컴포넌트.
- `HairstyleComponent`: 헤어스타일 이미지를 보여주고 다운로드 및 삭제를 할 수 있게 해 주는 컴포넌트.
- `HeaderComponent`: 사이트의 헤더 컴포넌트.
- `ImageBoxComponent`: 사용자가 입력하는(파일탐색기, 웹캠) 이미지를 표시하는 컴포넌트.
- `UserChatComponent`: 채팅페이지에서 사용되는 유저의 말풍선 컴포넌트입니다.
- `WebCamModalComponent`: 웹캠을 찍을 수 있는 모달창 컴포넌트.
